## Aplikasi KPU Provinsi

Aplikasi ini menggunakan Laravel sebagai frameworknya.
Penjelasan selanjutnya ntar aja kalo aplikasinya jadi.

### Instalasi
 - Download ZIP => Extract
 - `composer install`
 - `php artisan serve`
 - Langsung kunjungi `localhost:8000` melalui browser
 - `SELESAI`

### Akun Admin
 - Username : `admin`
 - Password : `admins`
